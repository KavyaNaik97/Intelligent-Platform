AIB Platform

AIB Platform is a next-generation Platform Agnostic and Cloud Agnostic framework which provides seamless integration of OEM and third party applications.

It provides access to hardware and other systems services using AIB API interface.

AIB Platform has dedicated AIB play store, from which user can install the required applications. All the apps are authenticated by AIB Security which includes confirmation of static and dynamic permissions. It is capable of providing dynamic configuration of interaction between different apps, avoiding core software update with 3-level world-class security. The data is encrypted at Rest and in Transit.

AIB Platform also provides Action Oriented Intelligent Behavioral Analysis (AOIBA).
